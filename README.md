# 视频逐帧AI分析工具

一个基于Web的应用程序，可自动上传视频、逐帧提取画面，并调用火山引擎AI大模型为每一帧生成详细的图像描述提示词。

## ✨ 核心功能

- **视频处理**：支持拖放或选择MP4、AVI、MOV、WebM格式视频（≤500MB），按可调帧率提取画面帧。
- **AI智能分析**：集成火山引擎视觉理解大模型（如豆包系列），为每帧图像自动生成结构化描述。
- **交互式结果展示**：以网格卡片展示每帧图像、时间戳及对应的AI提示词，支持展开/收起、一键复制提示词。
- **数据导出**：支持将所有帧的分析结果导出为JSON或CSV文件，便于后续处理。

## 🛠️ 技术栈

- **前端**：HTML5, CSS3, JavaScript (ES6+)
  - 视频帧提取：Canvas API
  - HTTP客户端：Fetch API
- **后端**：Node.js, Express
  - 安全代理：处理火山引擎API认证与请求转发
- **AI服务**：火山引擎方舟平台（视觉理解大模型）
